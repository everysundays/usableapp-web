## ğŸ¯ What Makes This Different from v1

### Avoiding from v1:
- No Lovable platform dependency
- No WordPress integration complexity
- No heavy component libraries (shadcn/ui, Radix UI)
- No bilingual language switching
- No PDF generation
- No video backgrounds (unless minimal/purposeful)
- No earth-tone color palette (cream/terracotta)
- Simpler, focused architecture

### New Direction:
- Single-page focus on core value proposition
- Problem-first messaging ("Good problems make good apps")
- Business card flip interaction for engagement tracking
- Clean, minimal aesthetic (inspired by iodigital.com)
- Fresh color palette based on concepts, not trends
- Anakotmai typography (Thai-optimized, modern)
- Fast, performant static site (Astro)
